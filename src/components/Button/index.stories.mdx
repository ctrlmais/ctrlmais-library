# Button
> O **_`<Button>`_** é um componente de botão que pode ser usado para ações de clique.

## _Props_

> Propriedades comuns do componente.

| Prop | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| `size` | `string` | `'md'` | Define o tamanho do botão, pode ser `sm`, `md` e `lg` |
| `children` | `string` | `undefined` | Define o texto do botão |
| `onClick` | `function` | `undefined` | Define a função que será executada ao clicar no botão |
| `textColor` | `string` | `undefined` | Define a cor do texto do botão |
| `backgroundColor` | `string` | `undefined` | Define a cor de fundo do botão |
| `hoverColor` | `string` | `undefined` | Define a cor de fundo do botão ao passar o mouse por cima |
| `fontSize` | `string` | `undefined` | Define o tamanho da fonte do botão |
| `fontWeight` | `string` | `undefined` | Define o peso da fonte do botão |

## _Observação_

> A Propriedade `full` irá esticar o máximo que puder dentro do container, porém limitada até `400px`
