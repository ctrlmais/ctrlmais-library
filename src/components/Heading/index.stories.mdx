# Heading
> O _**`<Heading>`**_ é um componente que pode ser usado para exibir cabeçalhos.

## _Props_

> Propriedades comuns do componente.

| Prop | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| `children` | `ReactNode<string>` | `undefined` | Conteúdo do heading |
| `size` | `string` | `md` | Tamanho do heading |

<Meta title="Docs/Heading" component={ Heading } />

export const Template = (args) => <Heading {...args} />;
