# Checkbox
> O _**`<Checkbox>`**_ é um componente que pode ser usado para ações de controle de estado.

## _Props_

> Propriedades comuns do componente.

| Prop | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| `children` | `ReactNode<string>` | `undefined` | Conteúdo do checkbox |
| `checked` | `boolean` | `false` | Define se o checkbox está selecionado ou não |
| `onCheckedChange` | `function` | `() => {}` | Função que será executada quando o checkbox for clicado |
| `backgroundColor` | `string` | `#1a1a1a` | Define a cor de fundo do checkbox |
| `labelColor` | `string` | `#C4C4CC` | Define a cor do texto do checkbox |
| `indicatorColor` | `string` | `#81D8F7` | Define a cor do indicador do checkbox |
