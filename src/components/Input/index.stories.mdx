# Input

> O **_`Input`_** é um componente moderno, que permite a entrada de dados pelo usuário.

## _Props_

> Propriedades comuns do componente.

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `placeholder` | `string` | `johndoe@gmail.com` | Placeholder do input |
| `type` | `string` | `text` | Tipo do input |
| `size` | `string` | `md` | Tamanho do input <u>_sm, md e lg_</u> |
| `value` | `string` | `null` | Valor do input |
| `onChange` | `function` | `() => {}` | Função que será executada quando o valor do input for alterado |
| `backgroundColor` | `string` | `#202024` | Cor de fundo do input |
| `placeholderColor` | `string` | `#7c7c8a` | Cor do placeholder do input |
| `fontColor` | `string` | `#000` | Cor da fonte do input |
