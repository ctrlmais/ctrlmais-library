# Label
> O _**`<Label>`**_ é um componente que pode ser usado para exibir textos categóricos.

## _Props_

> Propriedades comuns do componente.

| Prop | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| `children` | `ReactNode<string>` | `undefined` | Conteúdo da label |
| `size` | `string` | `md` | Tamanho da label |

<Meta title="Docs/Label" component={ Label } />

export const Template = (args) => <Label {...args} />;
